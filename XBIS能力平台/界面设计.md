# 🎨 UI风格建议（你现在必须统一）

## 颜色（建议）

```text
主色：#6366F1（AI感）
成功：#10B981
错误：#EF4444
背景：#F9FAFB
卡片：白 + 阴影
```

------

## 布局

```text
最大宽度：1200px
卡片间距：16px
圆角：12px
阴影：轻
```

------

## 按钮

```text
主按钮：渐变 or 主色
hover：加深
disabled：灰
```

------

# 🚀 最关键交互（你一定要实现）

## 能力执行

```text
填参数
→ 点击执行
→ 弹价格确认
→ 扣费
→ 创建任务
→ 显示 loading
→ 展示结果
```
