## 修改建议

### 建议1: 明确图表库选择（必须修改）
推荐选择 **Recharts**，理由：
- React 原生支持，API 友好
- 体积小，Tree-shaking 友好
- 与 Ant Design 风格一致

### 建议2: 补充配额警告逻辑（必须修改）
在 QuotaUsage 组件中增加：
```typescript
const warningThreshold = 0.8;
const dangerThreshold = 0.95;

// usagePercentage > 80% 显示警告
// usagePercentage > 95% 显示危险
```

### 建议3: 增加用量明细排序/筛选（建议修改）
UsageTable 支持：
- 按调用次数排序
- 按费用排序
- 按能力名称筛选

### 建议4: 明确多币种展示方案（建议修改）
在页面右上角增加币种选择器（CNY/USD），切换后重新加载数据。
