**严重程度: 低**

## 修改建议

### 建议1: 确定图表库选择（必须修改）
推荐选择 **ECharts**，理由：
- 功能丰富，支持多种图表类型
- 中文文档完善
- 支持暗色主题
- 按需引入支持 Tree-shaking

### 建议2: 补充告警已读 API（必须修改）
```typescript
// POST /admin-api/v1/overview/alerts/:id/read
// 标记单条告警已读

// POST /admin-api/v1/overview/alerts/read-all
// 标记所有告警已读
```

### 建议3: 明确不同数据的刷新策略（建议修改）
```typescript
const refreshStrategies = {
  stats: { interval: 300000, label: '5分钟' },      // 统计数据 5 分钟
  alerts: { interval: 60000, label: '1分钟' },       // 告警信息 1 分钟
  recentJobs: { interval: 30000, label: '30秒' },    // 最近任务 30 秒
};
```

### 建议4: 增加数据导出功能（建议修改）
支持导出：
- 统计报表（CSV/PDF）
- 最近任务列表（CSV）
- 告警记录（CSV）
