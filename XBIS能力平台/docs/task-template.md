# XBIS 开发任务卡模板

> 版本：v1.0  
> 用途：标准化功能开发任务，确保需求传递完整、开发过程可追溯、验收标准清晰

---

## 如何使用本模板

1. 复制本模板到 `tasks/` 目录，文件命名为 `feature-{功能标识}.md`
2. 填写所有必填字段（标记为 ⭐）
3. 技术评审前必须完成「数据结构」和「API 接口」章节
4. 开发完成后对照「验收标准」逐项自测
5. 联调前确认「是否影响现网」和「依赖任务」状态

---
