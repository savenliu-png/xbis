请基于当前任务卡和系统环境执行：

1. 启动服务并检查环境

   node manage-services.mjs start
   node manage-services.mjs stop
   node manage-services.mjs restart
   node manage-services.mjs status

2. 执行真实功能测试

3. 发现问题并分类（前端/后端/API/DB/数据）

4. 自动修复问题（必须给代码/SQL/API）

5. 执行联调

6. 执行回归测试

7. 输出验收结果

如果未通过：
→ 自动继续修复循环

直到：
→ 验收通过为止

输出每一轮：

- 问题列表
- 修复内容
- 测试结果
- 验收结论
- 是否继续
