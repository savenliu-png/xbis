# 🎯 目标（非常重要）

对一个“功能模块”执行：

# ❗系统级自动修复闭环（不是单任务）

模块包含：

- 多个前端任务
- 多个后端接口
- 数据库表结构
- UI页面
- 权限逻辑
- API调用链

必须完成：

👉 模块级：

```text
模块扫描 → 任务识别 → 依赖分析 → UI系统统一 → 功能补全 → API补齐 → DB补齐
→ 服务启动 → 前后端联调 → 真实测试 → Bug修复 → 回归 → 验收
→ 不通过继续循环
```
